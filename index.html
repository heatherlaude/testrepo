JavaScript Window Object & Math Object

Window is the Global Object. The window object is created for each window that appears in the browser. A window can be the main window, a frame set or a tab.

The Math object is a predefinied JavaScript object that has a variety of properties and methods to use. It is used for mathmetical constants or to perform certain calculations

Commonly Used Properties

  • window.innerHeight - Holds values for the height of the area of the window in view.
  • window.innerWidth - Holds values for the width of the area of the window in view.
  • window.location - Holds the current URL of the window.

    • Commonly Used Methods

    • alert() - Pops up a message - user has to click "OK".
    • prompt() - Prompts user to enter information.
    • confirm() - Pops up a window that requires user to confirm or cancel an action.
    • print() - Opens the print dialog.
